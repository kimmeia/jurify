ALTA

Exportar + importar no mesmo escritório: o cenário já é JSON estruturado por dentro. É serializar/desserializar + validar (já temos validação de grafo e dos tipos).

MÉDIA

Importar de outro escritório / template público: o fluxo referencia agente de IA, campos, responsável, Kanban, Cal.com — que mudam de escritório pra escritório. Precisa da tela de "resolver dependências" acima (o que o n8n faz com credenciais).

Recomendação: começar por Exportar (rápido, já ajuda em backup/compartilhar) + Importar no mesmo escritório. A reconciliação de dependências (cross-escritório / galeria de templates) vem como fase 2.
