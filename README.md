# JuridFlow

O sistema operacional do escritório de advocacia: gestão de processos, clientes
(CRM), financeiro, atendimento omnichannel, automações (SmartFlow), agentes de IA
e cálculos jurídico-financeiros.

> O repositório se chama `jurify` no GitHub por motivo histórico — o produto é **JuridFlow**.

Consulte `CLAUDE.md` para comandos, convenções e o fluxo de deploy.
